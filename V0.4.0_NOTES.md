# PTF SMM Bot v0.4.0 — Approval + Follow-up Operational Loop

## Added
- Multi-message campaign replies: summary, schedule, user tasks, post-event tail, visuals/media, drafts.
- Inline Telegram buttons for approval and posting status.
- Callback handling for Approve, Edit, Posted, Not Yet, Postpone, Skip, Regenerate.
- Approval Queue and User Decisions tracking.
- Reminder / Follow-up scheduler.
- Telegram Media Intake for photos/images sent directly to the bot.
- Style Pack / Reference Assets storage for poster references and player photos.
- Campaign State, Draft Versions and Asset Usage History tables.

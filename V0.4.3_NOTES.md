# v0.4.3 Visual-only hotfix

Fixes the router/orchestrator issue where poster generation requests were treated as full campaign creation.

Adds:
- visual-only detection before AI Router
- /generate_visual and /poster passthrough commands
- direct visual-only generation from active campaign
- attempts to use recent Telegram reference images via OpenAI image edit, with prompt-only fallback
- uploads incoming Telegram image references to Google Drive during media intake
- updates reference asset type when clicking Style / Player / Event buttons

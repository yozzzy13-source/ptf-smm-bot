# v0.5.7 Image Generation Delivery Fix

Changed files only.

- Generated images no longer fail just because Google Drive upload is unavailable/service-account quota blocks upload.
- SAVE_GENERATED_IMAGES_TO_DRIVE=false by default; images are sent to Telegram directly as file buffers.
- Added Telegram multipart photo upload from buffer.
- Visual-only reply now shows real generation errors instead of only “Generation Failed”.
- Reference collector now supplements from recent active references instead of using only one event-bound item.
